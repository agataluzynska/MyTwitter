# MyTwitter
Console twitter


Requirments:
Java 8
Maven

Command to run: java -jar target/MyTwitter-0.0.1-SNAPSHOT-jar-with-dependencies.jar

Example usage:
    posting: <user name> -> <message>
    reading: <user name>
    following: <user name> follows <another user>
    wall: <user name> wall
    
To quit press CTRL + C /CTRL + D
